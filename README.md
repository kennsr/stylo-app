# stylo

Simple fashion online shop with AI try-on feature

## Running the App

### Mock Mode (Default)

The app runs in mock mode by default, which means it uses local mock data instead of making real API calls. This is useful for UI development and testing.

To run the app in mock mode:

```bash
flutter run
# or explicitly:
flutter run --dart-define=ENV=mock
```

To run against other environments (if available):

```bash
flutter run --dart-define=ENV=staging
flutter run --dart-define=ENV=production
```
